# 🐾 Curious dog

Acesse [aqui](https://curious-dog.netlify.app/)

Projeto desenvolvido para aprender a consumir uma API externa e manipular os dados retornados com JavaScript.

## 🎯 Objetivo

Praticar os conceitos de:
- Requisição HTTP com `fetch`
- Tratamento de resposta em JSON
- Manipulação do DOM com os dados recebidos

## 🔌 API utilizada

[Dog API](https://dogapi.dog/) — API pública e gratuita que retorna curiosidades sobre cães.

**Endpoint usado:**
```
GET https://dogapi.dog/api/v2/facts?limit=1
```

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
